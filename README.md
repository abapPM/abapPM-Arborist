# apm Arborist

Inspect and manage package trees.

apm is a *package manager* 📦 for ABAP applications and modules, a *website* 🌐, and a *registry* 📑.

You can find the *website* at https://abappm.com and the *registry* at https://registry.abappm.com.

NO WARRANTIES, [MIT License](https://github.com/abapPM/abapPM-Arborist/blob/main/LICENSE)

## Usage

Not documented yet

## Prerequisites

SAP Basis 7.50 or higher

## Installation

Install `@apm/arborist` as a global module in your system using [apm](https://abappm.com).

or

Specify the `@apm/arborist` module as a dependency in your project and import it to your namespace using [apm](https://abappm.com).
